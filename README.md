# thoughtground

A live playground showcasing the capabilities of [react-thoughtcast](https://github.com/monkkeyking/react-thoughtcast). No features, no opinions — just the library doing its thing.

## Purpose

This project exists solely to demonstrate what react-thoughtcast can do. It is not a production app and not a template. If you want to see the library in action before using it, this is the place.

## AI friendly

This codebase is intentionally kept simple and well-structured so that AI tools (Claude, Copilot, etc.) can read, modify, and extend it without friction. Each card is an isolated component, constants are centralized, and there is no shared state between unrelated parts.

Feel free to paste any file into your AI tool of choice and ask it to add new demos, tweak animations, or experiment with the API.

## Stack

- React 19
- react-thoughtcast 
- Vite + TypeScript

## Run locally

```bash
npm install
npm run dev
```
